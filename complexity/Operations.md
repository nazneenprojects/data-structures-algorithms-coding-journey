
# Data Structure & Operations

## Array

| Operation   | Time Complexity | Space Complexity |
|:------------|:--------------:|-------------:|
| Reading     | O(1)           |        |
| Insertion    | O(n)          |        |
| Deletion    | O(n)          |        |


## LinkedList

| Operation   | Time Complexity | Space Complexity |
|:------------|:--------------:|-------------:|
| Reading        | O(n)         |        |
| Insertion        | O(1)         |        |
| Deletion        | O(1)         |        |
